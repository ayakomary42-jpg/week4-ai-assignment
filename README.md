## Question 1: What is data?
Data refers to raw facts, figures and information that can be collected, stored and analyzed to produce meaningful insights.
These data may not have meaning until it is processes or interpreted.
Examples:
1. The number of students in a class (e.g 60)
2. A person's age or name stored in a system
## Question 2: Difference between structured and unstructured data
### Structured data
Refers to data that follows a strict schema which makes it easier to store, search and analyze data.
Examples:
1. Aspreadsheet containing student names, admission numbers, and grades
2. A database table containing customer IDs, phone numbers, and adresses
### Unstructured data
Refers to data that does not follow a strict schema and is usually harder to organize and analyze.
Examples:
1. Images and videos
2. Emails or PDFs
3. Social media posts
## Question 3: What is schema?
A schema is the structure that defines how data is organized in a database
It can describe the tables, fields, relationships, and rules for storing data
Example:
Consider a spreadsheet used to store the following student information 

| Admission Number | Student Name | Course | Grade |
|------------------|-------------|--------|-------|
| 1001             | Alice       | AI     | A     |
| 1002             | Brian       | Data Science | B |
| 1003             | Carol       | Cyber Security | A |
In this example, the **schema** defines that every student record must contain:
- Admission Number  
- Student Name  
- Course  
- Grade  
Each row represents one student's data, while the column names define the **structure of the data**.
## Bonus Question: Difference between list and dictionary
### List
A list is an ordered collection of items and each item can be accessed using its position (index).
Example:
["Python", "Java", "C++"]
### Dictionary
A dictionary stores data in key-value pairs and this each value is accessed using a unique key.
Example:
{
"name": "Mary",
"age": "22"
}
